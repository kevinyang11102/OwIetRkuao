# 人员管理系统简介 python395

## 项目概述

人员管理系统是基于Pyqt的Python模板程序，旨在提供一套完善的人员信息管理解决方案。系统包含人员信息的录入、修改、删除和查找等功能模块。

## 技术栈

- **后端**：Python
- **前端**：Pyqt

## 功能模块

- **人员信息管理**
  - 录入
  - 修改
  - 删除
  - 查找

## 系统角色

- 管理员：负责系统操作和人员信息管理

## 运行环境

- Python环境
- Pyqt库
